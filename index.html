7) Triángulo con vértices A(1,1), B(0,5), C(-3,0)

Comprobación de que es rectángulo:

Método A (Teorema de Pitágoras):

AB² = (0-1)² + (5-1)² = 1 + 16 = 17

AC² = (-3-1)² + (0-1)² = 16 + 1 = 17

BC² = (-3-0)² + (0-5)² = 9 + 25 = 34

AB² + AC² = 17 + 17 = 34 = BC² → Es rectángulo

Método B (Pendientes):

m(AB) = (5-1)/(0-1) = 4/(-1) = -4

m(AC) = (0-1)/(-3-1) = (-1)/(-4) = 1/4

m(AB) × m(AC) = -4 × 1/4 = -1 → Rectas perpendiculares

Conclusión: El triángulo es rectángulo con ángulo recto en A(1,1)

1) Hallar las coordenadas del punto Q que divide al segmento Q1(-2,3) y Q2(3,-2) en razón r=2/5

Paso 1. Fórmula de división interna:

P(x,y) = ((x₁ + r·x₂)/(1 + r), (y₁ + r·y₂)/(1 + r))

Paso 2. Cálculo de la coordenada x:

x = (-2 + (2/5)·3) / (1 + 2/5)

x = (-2 + 6/5) / (7/5)

x = (-10/5 + 6/5) / (7/5)

x = (-4/5) / (7/5) = -4/7

Paso 3. Cálculo de la coordenada y:

y = (3 + (2/5)·(-2)) / (1 + 2/5)

y = (3 - 4/5) / (7/5)

y = (15/5 - 4/5) / (7/5)

y = (11/5) / (7/5) = 11/7

Resultado: Q(-4/7, 11/7)

P(x,y) = ((x₁ + r·x₂)/(1 + r), (y₁ + r·y₂)/(1 + r))

Paso 2. Cálculo de la coordenada x:

x = (8 + (-2)·2) / (1 + (-2))

Paso 3. Cálculo de la coordenada y:

1) Triángulo con vértices A(-2,-1), B(2,2), C(5,-2)

Área = ½ |x₁y₂ + x₂y₃ + x₃y₁ - (y₁x₂ + y₂x₃ + y₃x₁)|

Paso 2. Sustitución de valores:

Área = ½ |(-2)(2) + (2)(-2) + (5)(-1) - [(-1)(2) + (2)(5) + (-2)(-2)]|

Área = ½ |-4 -4 -5 - (-2 + 10 + 4)|

Área = ½ |-13 - 12| = ½ × 25 = 12.5 u²

Paso 3. Cálculo del perímetro:

AB = √[(2-(-2))² + (2-(-1))²] = √[4² + 3²] = √[16+9] = √25 = 5

BC = √[(5-2)² + (-2-2)²] = √[3² + (-4)²] = √[9+16] = √25 = 5

AC = √[(5-(-2))² + (-2-(-1))²] = √[7² + (-1)²] = √[49+1] = √50 = 5√2

Perímetro = 5 + 5 + 5√2 = 10 + 5√2 ≈ 17.0711 u

Resultado: Área = 12.5 u², Perímetro = 10 + 5√2 ≈ 17.0711 u

1) Ángulos interiores del triángulo A(-2,1), B(3,4), C(5,-2)

3) Dos rectas se cortan formando 135°; m₁ = -3. Hallar m₂

Paso 1. Ángulo agudo entre rectas:

Ángulo dado: 135° ⇒ Ángulo agudo = 180° - 135° = 45°

tan θ = |(m₂ - m₁)/(1 + m₁m₂)| = tan 45° = 1

Paso 2. Resolución de ecuaciones:

Caso 1: (m₂ - m₁)/(1 + m₁m₂) = 1 ⇒ m₂ - m₁ = 1 + m₁m₂

m₂(1 - m₁) = 1 + m₁ ⇒ m₂ = (1 + m₁)/(1 - m₁)

Caso 2: (m₂ - m₁)/(1 + m₁m₂) = -1 ⇒ m₂ - m₁ = -1 - m₁m₂

m₂(1 + m₁) = m₁ - 1 ⇒ m₂ = (m₁ - 1)/(1 + m₁)

Paso 3. Sustitución m₁ = -3:

Caso 1: m₂ = (1 + (-3))/(1 - (-3)) = (-2)/4 = -1/2

Caso 2: m₂ = (-3 - 1)/(1 + (-3)) = (-4)/(-2) = 2

Resultado: m₂ = 2 o m₂ = -1/2

m⊥ = -1/m₁ = -1/(2/7) = -7/2

Paso 3. Ecuación punto-pendiente:

y - 3 = -7/2(x - 2)

y - 3 = -7/2x + 7

y = -7/2x + 10

Paso 4. Intersecciones:

Eje x (y = 0): 0 = -7/2x + 10 ⇒ 7/2x = 10 ⇒ x = 20/7

Eje y (x = 0): y = -7/2(0) + 10 = 10

Paso 5. Ángulo de inclinación:

tan θ = m⊥ = -7/2 ≈ -3.5 ⇒ θ ≈ -74.05°

Ángulo positivo: 180° - 74.05° = 105.95°

Resultado: y = -7/2x + 10, Intersecciones: (20/7, 0) y (0, 10), Ángulo ≈ 105.95°

4) Recta que pasa por la intersección de 2x-3y-5=0 y x+2y-13=0, y segmento sobre eje x = doble de su pendiente

Paso 1. Punto de intersección:

Sistema: 2x - 3y - 5 = 0 y x + 2y - 13 = 0

De la segunda: x = 13 - 2y

Sustituyendo: 2(13 - 2y) - 3y - 5 = 0 ⇒ 26 - 4y - 3y - 5 = 0

21 - 7y = 0 ⇒ y = 3

x = 13 - 2(3) = 7

Punto de intersección: P(7, 3)

Paso 2. Condición del problema:

Segmento sobre eje x = 2 × pendiente

Para y = mx + b, intersección con eje x: x = -b/m

-b/m = 2m ⇒ b = -2m²

Paso 3. La recta pasa por P(7, 3):

3 = m(7) + b = 7m - 2m²

-2m² + 7m - 3 = 0 ⇒ 2m² - 7m + 3 = 0

Paso 4. Resolver ecuación cuadrática:

m = [7 ± √(49 - 24)]/4 = [7 ± √25]/4 = [7 ± 5]/4

m₁ = 12/4 = 3, m₂ = 2/4 = 1/2

Paso 5. Hallar b para cada pendiente:

Para m = 3: b = -2(3)² = -18

Para m = 1/2: b = -2(1/2)² = -2(1/4) = -1/2

Resultado: Dos rectas posibles:

y = 3x - 18

y = (1/2)x - 1/2

Distancia de un punto a una recta

1.Calcular la distancia de la recta 3x - 4y + 6 = 0 al punto A(7, -3)

Fórmula de distancia de un punto a una recta Ax + By + C = 0:

d = |Ax₀ + By₀ + C| / √(A² + B²)

Paso 1: Identificar los valores:

A = 3, B = -4, C = 6, x₀ = 7, y₀ = -3

Paso 2: Sustituir en la fórmula:

d = |3(7) - 4(-3) + 6| / √(3² + (-4)²)

Paso 3: Calcular el numerador:

3(7) = 21, -4(-3) = 12, 21 + 12 + 6 = 39

|39| = 39

Paso 4: Calcular el denominador:

√(3² + (-4)²) = √(9 + 16) = √25 = 5

Paso 5: Calcular la distancia:

d = 39 / 5 = 7.8

✅ Distancia del punto a la recta: d = 7.8 unidades

2) Distancia entre rectas paralelas

Calcular la distancia entre las rectas paralelas:

3x - 4y + 8 = 0 y 6x - 8y + 9 = 0

Paso 1: Verificar que son paralelas y poner en la misma forma:

Dividimos la segunda ecuación entre 2 para que los coeficientes de x e y coincidan:

6x - 8y + 9 = 0 ⇒ 3x - 4y + 9/2 = 0

Ahora tenemos:

L₁: 3x - 4y + 8 = 0

L₂: 3x - 4y + 4.5 = 0

Paso 2: Fórmula de distancia entre rectas paralelas Ax + By + C₁ = 0 y Ax + By + C₂ = 0:

d = |C₂ - C₁| / √(A² + B²)

Paso 3: Sustituir valores:

A = 3, B = -4, C₁ = 8, C₂ = 4.5

d = |4.5 - 8| / √(3² + (-4)²)

Paso 4: Calcular:

Numerador: |4.5 - 8| = |-3.5| = 3.5

Denominador: √(9 + 16) = √25 = 5

d = 3.5 / 5 = 0.7

✅ Distancia entre las rectas paralelas: d = 0.7 unidades

La ecuación de la recta como modelo matemático

Problema: El valor de una computadora disminuye con el tiempo. Se sabe que a los 10 años de uso vale $23,000 y a los 6 años de uso vale $41,400.

a) Encontrar la ecuación particular y = mx + b

Paso 1: Calcular la pendiente m:

m = (y₂ - y₁) / (x₂ - x₁) = (41400 - 23000) / (6 - 10)

m = 18400 / (-4) = -4600

✅ Pendiente: m = -4600 (representa la depreciación anual)

Paso 2: Hallar b usando el punto (10, 23000):

y = mx + b ⇒ 23000 = -4600(10) + b

23000 = -46000 + b ⇒ b = 23000 + 46000 = 69000

✅ Ecuación particular: y = -4600x + 69000

b) Valor cuando era nuevo (x = 0)

y = -4600(0) + 69000 = 69000

✅ El valor nuevo de la computadora era $69,000.

c) Depreciación por año

La pendiente m = -4600 indica que la computadora pierde $4,600 por año.

d) Valor después de 12 años (x = 12)

y = -4600(12) + 69000 = -55200 + 69000 = 13800

✅ Valor después de 12 años: $13,800

e) ¿Cuántos años cuando el valor sea $4,600?

4600 = -4600x + 69000 ⇒ -4600x = 4600 - 69000 = -64400

x = (-64400) / (-4600) = 14

✅ Se vendería a los 14 años de uso.

f) ¿Después de cuántos años se deprecia totalmente (y = 0)?

0 = -4600x + 69000 ⇒ -4600x = -69000 ⇒ x = 69000 / 4600 = 15

✅ La computadora se deprecia totalmente después de 15 años de uso.

Resumen final:

Inciso Resultado
a) Ecuación y = -4600x + 69000
b) Valor nuevo $69,000
c) Depreciación anual $4,600
d) Valor a 12 años $13,800
e) Años para $4,600 14 años
f) Años para depreciación total 15 años
